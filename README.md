# DrumModule

Drum module based on Daisy Seed, with one MIDI input and one audio output.
Light to build, fits everywhere, sounds cool on big sound soundsystem.
Kind of emulate some 808 sounds.

Soundlist with MIDI notes :

Maracas        42

Ride           51
